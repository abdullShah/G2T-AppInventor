# G2T (Gallery to Telegram)

Приложение на [MIT App Inventor](https://ai2.appinventor.mit.edu/) для пересылки медиа-файлов из галереи в мессенджеры и социальные сети.

![Главный экран](assets/mainScreen.jpg)

![Рабочий экран](assets/workScreen.jpg)

## Возможности

- выбор до 10 фото или видео из галереи;
- отображение даты создания каждого файла;
- автоматическое определение основной даты по первому выбранному файлу;
- подсветка файлов с той же датой другим цветом;
- отправка через системное меню «Поделиться» с текстом основной даты;
- интернет не требуется — приложение работает локально.

## Сценарий работы

1. Пользователь открывает приложение.
2. Нажимает кнопку **«Выбрать из галереи»** и выбирает одно или несколько медиа-файлов: фото или видео.
Максимум — 10 файлов.
3. На экране появляются выбранные файлы с датами создания.
Дата первого выбранного файла становится основной.
Все файлы с этой датой выделяются другим цветом.
При неверном выборе можно снова нажать **«Выбрать из галереи»**.
4. Пользователь нажимает кнопку **«Отправить»** и выбирает нужную социальную сеть или другое приложение, в которое можно переслать файлы вместе с текстом основной даты.

## Установка

1. Откройте [MIT App Inventor](https://ai2.appinventor.mit.edu/).
2. Выберите **Projects → Import project (.aia) from my computer** и укажите файл `G2T.aia`.
3. Для тестирования: **Connect → AI Companion** на Android-устройстве. Для установки на телефон: **Build → Android App (.apk)**.
